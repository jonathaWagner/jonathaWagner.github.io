Source

Desenvolver uma página web costuma ser trabalhoso quando não usamos ferrementas certas. Em softwares de codificação de páginas web, temos que escrever todo os componentes da página (Html), salvar, executar em algum navegador e modificar os componentes que estão se comportando de maneira indesejada. A aba Sources permite que você faça as próprias alterações na aba Elements facilitando o processo de desenvolvimento

Network

Páginas web são formadas com diversos componentes como vídeos, imagens , arquivos css. Quanto mais componetes tem a página maior vai ser o tempo gasto para carregá la, a aba Network ajuda o desenvolvedor com informações sobre o tempo gasto para carregar os componentes ajudando na performance da página
